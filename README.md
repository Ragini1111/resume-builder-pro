# Resume Builder Pro

Live demo: https://ragini-gupta.github.io/resume-builder-pro/

ATS-friendly 1-page resume builder for engineering students. No signup, no backend. Fill details → Download PDF. 

### Features
- **Live preview** - See changes instantly as you type
- **PDF export** - 1-click A4 download using html2pdf.js
- **3 themes** - Blue tech, PCB green, Minimal black
- **Auto-save** - Never lose data, stored in browser
- **ATS optimized** - Single column, no tables/images, 13px font
- **Dynamic sections** - Add unlimited projects/experience

### Why this exists
Built for Academy of Technology ECE students to replace Canva templates that fail ATS scans. Used by 200+ students for 2026 placements.

### Tech Stack
HTML, CSS, Vanilla JS, html2pdf.js

### Setup
Just open `index.html` in any browser. No npm, no build step.
